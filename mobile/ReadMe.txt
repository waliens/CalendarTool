CONTENT:

- app-debug.apk: open this file with a real Android device to install the application. You will then be able to test the application with no additional action required.

- The directory MyUlgCalendar: this is the whole directory of the application created by Android Studio. You can import it inside an IDE to launch the application on a virtual device.

The easiest way to test the application is to install it on a real device using the apk file provided. If you nevertheless want to test it on a virtual device, we recommend to use a virtual Google Nexus 5 with API 21 because this is the one we used for our testings.
