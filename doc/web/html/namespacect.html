[in]function$cmp_fnA comparison function for comparing the element indexed by $index This function must have the same specification as the one below
Return values
functionThe comparison function
Note
A comparison function (passed as argument and returned) takes two arguments (the two subarrays) and returns an integer :
  • < 0 : if the first element is less that the second
  • 0 : if the elements are equal
  • > 0 : if the first element is greater than the second
