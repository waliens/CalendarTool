Detailed Description

Base class for category model.

A set of categories is a list of items contained in a table of the database that can be given to the user in a drop down list for instance (can be colors, jobs,...). A category is identified by its key and defined by its name and possible some other attributes. This class provides methods for accessing, editing and interacting with these items.
