run (   $cmd,

)

Execute an external program.

This method uses as PDO like syntax to build commands.

Parameters
string$cmdCommand to execute. If you need to pass arguments to a command. A PDO like syntax can be used. For example: "ls -lsa %path".
array$argsAn associative array containing the arguments to pass to $cmd. For example: array('path' => '/home');
string$stdinIf the command requests input (e.g. a passphrase) this can be passed here. Note that if a command requires multiple feedbacks from a user this method can not be used.
Returns
array Returns an array containing return value and results from STDOUT and STDERR.
