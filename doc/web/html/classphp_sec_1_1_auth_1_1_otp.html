generate (   $action,

  $length = 6,

Generate a one-time-password (OTP). The password is only valid for a given time, and must be delivered to the user instantly. The password is also only valid for the current session.

Parameters
string$actionThe action to generate a OTP for. This should be as specific as possible. Used to ensure that the OTP is used for the intended action.
string$uidUser identifier. Default is a session identifier. Can be set to a username or user id if you want the OTP to be valid outside the session active when creating it.
array$dataOptional array of data that belongs to $action. Used to ensure that the action is performed with the same data as when the OTP was generated.
integer$lengthOTP length.
integer$ttlTime to live for the OTP. In seconds.
Returns
string One time password that should be delivered to the user by for example email or SMS.
