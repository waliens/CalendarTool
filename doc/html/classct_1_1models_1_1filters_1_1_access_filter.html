Detailed Description

A class for filtering event according to user rights (can a user see this event?)

Constructor & Destructor Documentation

__construct (   $policy = null)

Construct the AccessFilter.

Parameters
[in]int$policyThe policy for selecting events (a combination of POLICY_* flags) (optionnal, default:the one that suits the most the user)
Note
The flags POLICY_AS_STUDENT and POLICY_AS_TEACHING_STUDENT cannot be combined with the POLICY_AS_FACULTY_MEMBER flag
Default policy is given by the static functions get_***_def_policy according to the user type
