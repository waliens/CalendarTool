Parameters
[in]string$startA datetime in the french format
[in]string$endA datetime in the french format (optional)
[in]int$modeThe selection mode (optionnal, default: see notes)
Note
The $mode is set to self::MODE_BETWEEN if an end date is given. If only a start date is given, then the default mode is self::MODE_AFTER and cannot be self::MODE_BETWEEN
