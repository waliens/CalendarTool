get_event_temporal_data (   $event_id,
  $lock_mode = Model::LOCKMODE_NO_LOCK 
)

Returns the temporal data of the given event.

Parameters
[in]int$event_idThe identifier of the event
[in]int$lock_modeOne of the Model LOCKMODE_* class constant
Return values
arrayArray containing the temporal data (empty array means that the event does not exist)
Note
The array contains a field 'Type' of which the value is one of the TEMP_* class constant. This field identify the temporal type of the event
This function accesses the tables time_range_event, date_range_event and deadline_event
According to the type the event is structured as follows :
  • deadline : array('End' => datetime, 'Type' => ...)
  • time_range : array('Start' => datetime, 'End' => datetime, 'Type' => ...)
  • date_range : array('Start' => date, 'End' => date, 'Type' => ...)
