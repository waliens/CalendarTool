get_sub_lockmode (   $lock_mode)
protected

As the unlock can only be performed when the complete set of operation was performed, then a function that receives a LOCKMODE_UNLOCK lock mode and call other function that should acquire lock cannot propagate this lock mode to these function. This function returns the lock mode that should be propagated these functions.
